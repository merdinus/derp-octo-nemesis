This is an example of a blockquote which is not inline, and is long!
I'm mostly using these drawn-out bits of speech to test out the functionality of the paragraph function, activated by the use of the tag < p > (minus the spaces, of course) and closed by the tag < / p >.

Of course, quoting html code like that is going to get annoying after a while, so the way round that little hiccup is to use the character entities recognised by many browsers. In this way, I can display the left code bracket as a less-than sign, signified by & l t ;, which displays as <. The other end of this is & g t ;, which displays as >.

I can also make this little bit of code identify itself to the casual reader by tricky application of the <acronym> function. Check this out: I need to <p>.

Now I know this trick, fear hovering above any of my text, EVAR.
